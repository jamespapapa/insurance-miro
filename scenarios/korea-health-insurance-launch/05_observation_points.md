# 결과 해석 체크포인트

## 꼭 볼 것

### 1. 전체 톤
- 최종 여론이 전반적으로 호기심 우위인지
- 아니면 불신/비판 우위인지

### 2. 주요 우려 포인트 우선순위
아래 중 무엇이 가장 크게 나오는지:
- 보험료 부담
- 보장 실효성 의심
- 약관 복잡성
- 개인정보/데이터 연동 우려
- 기존 보험 대비 차별성 부족

### 3. 채널별 온도차
- X/속보형 채널: 어떤 프레임으로 이야기하는지
- 커뮤니티형 채널: 어떤 후기/분석/비꼼이 퍼지는지
- 설계사 채널: 어떤 논리로 판매 혹은 방어하는지

### 4. 전환 트리거
- 긍정에서 부정으로 돌아서는 계기
- 부정에서 재평가되는 계기
- 커뮤니티 글/기사/비교표/후기 중 무엇이 제일 큰 영향력을 가졌는지

### 5. 실행 인사이트
최종적으로 아래 형태로 정리되면 좋다.

- 유지할 메시지 3개
- 삭제/축소할 메시지 3개
- 소비자 FAQ로 먼저 선제 대응할 질문 5개
- 출시 전 보완이 필요한 상품 구조 3개

## 다음 라운드 고도화용 추가 문서

첫 테스트 후 아래 자료를 추가하면 품질이 확 올라간다.

- 실제 보험 약관 요약본
- 경쟁사 건강보험 비교표
- 보험 커뮤니티 반응 캡처
- 소비자 민원 사례
- 건강보험/실손보험 관련 최근 뉴스 10~20건
