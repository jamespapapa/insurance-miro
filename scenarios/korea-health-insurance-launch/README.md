# 한국 보험시장 건강보험 신상품 출시 여론 시뮬레이션 스타터셋

빠르게 MiroFish에서 돌려볼 수 있도록 만든 **테스트용 시드 세트**다.

## 목적

한국 금융시장, 특히 보험시장 맥락을 반영한 상태에서
**새 건강보험 상품 출시 시 소비자/커뮤니티/미디어/설계사 채널 반응**을 시뮬레이션한다.

## 포함 파일

- `01_market_context.md`
  - 한국 금융시장/보험시장 배경
- `02_product_brief.md`
  - 가상의 건강보험 상품 콘셉트
- `03_public_sentiment_map.md`
  - 예상 반응축, 확산 포인트, 논쟁거리
- `04_simulation_requirement.txt`
  - MiroFish 입력창에 그대로 붙여넣을 수 있는 시뮬레이션 요구문
- `05_observation_points.md`
  - 결과 볼 때 체크할 질문/KPI

## 권장 사용법

### 1) 업로드할 문서
다음 3개를 업로드:
- `01_market_context.md`
- `02_product_brief.md`
- `03_public_sentiment_map.md`

### 2) 시뮬레이션 요구문
입력창에 `04_simulation_requirement.txt` 내용을 그대로 붙여넣기.

### 3) 결과 해석
최종 리포트/인터랙션 단계에서 `05_observation_points.md` 기준으로 확인.

## 주의

- 이 세트는 **빠른 프로토타입 검증용**이다.
- 실제 의사결정용으로 쓰려면 아래 자료를 추가로 넣는 게 좋다:
  - 금융감독원/생보협회/손보협회 공개자료
  - 경쟁사 건강보험 약관/보장요약서
  - 최근 보험 민원/불완전판매 이슈 정리
  - 커뮤니티 반응 캡처/기사/유튜브 스크립트

## 채널 해석 팁

MiroFish는 소셜/커뮤니티 시뮬레이션 성격이 강해서,
이번 시나리오에서는 채널을 아래처럼 해석하면 된다.

- Twitter/X 계층:
  - 속보형 여론, 기자/인플루언서/정치·경제 논객 반응
- Reddit 계층:
  - 한국형 커뮤니티 레이어로 해석
  - 예: 맘카페, 블라인드, 디시, 클리앙, 네이버카페, 보험설계사 커뮤니티
